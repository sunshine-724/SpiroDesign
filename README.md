# SpiroDesign(プロジェクト演習G1)  
## このブランチ(develop-desktop)は主にデスクトップ用PC向けのアプリ開発を行います。  
ブランチ名の規則は以下の通りです。  
feature/xxx             ← 新機能開発用  
release/xxx             ← リリース準備用  
hotfix/xxx              ← 緊急修正用  

## Javadocの生成方法

以下のコマンドを実行することで、Javadoc を生成できます。

./gradlew javadocSpiroDesign

出力された Javadoc は以下のディレクトリに保存されます。

build/docs/javadocSpiroDesign
